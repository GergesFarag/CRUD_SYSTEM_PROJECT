
# CRUD MANAGEMENT SYSTEM

The CRUD Management System is a basic web-based application that allows users to (create, read, update, and delete) records from a database. The system can be used to manage entities such as users, products, or other data structures.

This project serves as a template or starting point for building more complex web applications with full CRUD functionality.


## Technologies Used

**Front-end :** Html , Css , Js , bootstrap

**back-end :** NodeJS , ExpressJS , MongoDB 

**ORM/Database Driver :** Mongoose


## Run Locally

Clone the project

```bash
  git clone https://link-to-project 
```
**OR Download .Zip file**

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node index.js
```
## System Features : 
- JWT Authentication in (Login - Register - Logout )
- Authentication and validation on user inputs
- Controlled MVC Architecture
- Responsive design via bootstrap
